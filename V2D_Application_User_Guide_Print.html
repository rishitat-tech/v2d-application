V2D Application User Guide

A guide for using V2D to access MV, EGO, Object Scanning, Dashboard User Guide, and quiz resources in one organized place.

Use For Training resources, guides, examples, and quizzes.
Main Areas MV, EGO, Object Scanning, Dashboard Guide, Quiz.
Resource Types Documents, videos, examples, checklists, FAQs, images, and 3D models.
Quiz Results Scores are saved in the browser and can be downloaded as CSV.

Overview

V2D is a resource hub for MV, EGO, Object Scanning, Dashboard User Guide, and training quizzes. Use this application to find guides, videos, examples, checklists, FAQs, and quizzes in one place.

V2D Resource Hub

The Resource Hub is the main place to access all training and reference materials.

  • MV
  • EGO
  • Object Scanning
  • Dashboard User Guide
  • Quiz
Use the search box to quickly find a section or resource.

MV Section

The MV section contains resources related to MV workflows.

Overview

High-level MV workflow information and data collection overview.

Videos

MV training, calibration, output visualization, and example videos.

Docs

Detailed MV documentation, calibration guide, and data collection guidance.

Examples

Good and bad examples to understand quality expectations.

Use the Checklist tab for quick reminders and the FAQ tab for common MV questions.

EGO Section

The EGO section contains resources related to EGO recording workflows.

Overview

High-level EGO workflow, operator checks, and data collector expectations.

Videos

EGO good and bad recording examples.

Docs

EGO documentation, data collector guide, and operator guide.

Examples

Good and bad examples showing recording quality expectations.

Use the Checklist tab for pose, visibility, and movement reminders. Use the FAQ tab for common EGO questions.

Object Scanning Section

The Object Scanning section contains resources for scanning workflows.

Overview

High-level object scanning workflow information.

Videos

Scanning tutorial videos when available.

Docs

Documentation and external links for scanning tools and workflows.

3D Examples

Good and bad GLB model examples display directly in the card.

Use the Checklist tab before and during scanning. Use the FAQ tab for common object scanning questions.

Dashboard User Guide

The Dashboard User Guide section provides access to the dashboard guide. Use this section to open or download the dashboard documentation.

The guide explains how to use the operations dashboard for daily updates, MV/EGO stations, inventory, orders, station issues, session planning, data collectors, and reports.

Quiz Section

The Quiz section contains training validation quizzes.

  • MV Quiz
  • EGO Quiz
  • Object Scanning Quiz

Taking a Quiz

  1. Open the Quiz section.
  2. Select the quiz you want to take.
  3. Enter your name and email.
  4. Answer every question.
  5. Click Submit Quiz.
  6. Review your score and answers.

Saved Quiz Results

The Saved Results table shows quiz name, name, email, score, percentage, and submitted time.

Quiz results are saved in the current browser only. Use Download CSV to export saved results.

Searching and Opening Resources

Each resource section includes a search box. Search by title, description, or tags.

  • Documents open in a new tab.
  • Videos open in the in-app video player.
  • Images display directly in the card.
  • 3D models display directly in the card.
  • Checklist and FAQ cards display information directly.
If a media file has not been added yet, the card may show “Media file not added yet.”

Recommended Usage Flow

  1. Start at the Home page.
  2. Open the V2D Resource Hub.
  3. Select MV, EGO, or Object Scanning.
  4. Review the Overview tab.
  5. Open the Docs tab for detailed instructions.
  6. Review Good Examples and Bad Examples.
  7. Use the Checklist before starting work.
  8. Take the related quiz.
  9. Download quiz results if needed.

Notes

  • Use guides for detailed instructions.
  • Use examples to understand quality expectations.
  • Use checklists as quick reminders.
  • Use quizzes to confirm understanding.
  • Quiz results are stored locally in the browser.
