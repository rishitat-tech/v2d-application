EGO Operator Guide

This guide explains what the operator should check during EGO recording and includes the commands used to start camera recording with the pre-built bundle.

Before Recording

  • Confirm the head-mounted camera view is clear.
  • Confirm hands and wrists are visible.
  • Confirm the task area is visible.
  • Make sure no extra person or extra hand is visible in frame.
  • Confirm the data collector is ready to hold the start pose for 3 seconds.

Starting the Camera Recording

Activate Environment

At the root directory of egocam_collection_bundle/, run:

uv venv --python 3.11 venv_isaacteleop
source venv_isaacteleop/bin/activate

Install Dependencies

uv pip install -r requirements.txt

Run with Preview and Upload

uv run egocam_collection.py --bucket-path s3://recordings/v2d/ego/ --endpoint-url https://pdx.s8k.io --access-user '' --secret-key '' --region us-east-1 --preview

Run with Preview and No Upload

uv run egocam_collection.py --no-upload --preview
The script prompts for a session ID. Leaving it empty uses the timestamp as the ID. It also prompts for stream selection. Leaving stream selection empty selects Color.

Stopping Recording

Press:

Ctrl + C

A raw H264 frame file and MP4 mux for each stream will be saved under: 

ego-reconstruction/collection/recordings/

Naming convention:

SessionID_{color|monoleft|monoright}.{h264|mp4}

During Recording

  • Watch that the task interaction stays in frame.
  • Watch that hands, wrists, and objects remain visible.
  • Check that the data collector moves slowly and naturally.
  • Check that the video remains stable.
  • Watch for fast, jerky, or unrelated movements.
  • Confirm the data collector returns to the start pose at the end.

Redo Rules

  • If hands go out of frame, delete the sequence and redo it.
  • If wrists go out of frame, delete the sequence and redo it.
  • If the object or main task interaction goes out of frame, delete the sequence and redo it.
  • If another person or extra hand appears in frame, delete the sequence and redo it.
Do not keep a sequence if important task interaction is missing from the head-mounted camera view.

After Recording

  • Confirm the task is complete.
  • Confirm the data collector returned to the start pose at the end.
  • End the recording.
  • Confirm whether the sequence is usable.
