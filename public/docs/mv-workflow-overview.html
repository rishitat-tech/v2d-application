MV Workflow Overview

This overview explains the main MV workflow at a high level. Use the full MV documentation for complete details and the separate guides for calibration and data collection steps.

Purpose

MV data collection is used to capture controlled task sequences with the data collector and target object visible inside the marked bounding area. The goal is to collect clean, usable, high-quality data.

Main Workflow

  1. Prepare the cameras and equipment.
  2. Complete calibration and verify the calibration video quality.
  3. Select the required pre-scanned object.
  4. Confirm the data collector is dressed correctly and badge is removed.
  5. Confirm the data collector and object are inside the marked bounding area.
  6. Operator starts recording and gives the countdown.
  7. Data collector performs the task slowly, naturally, and consistently.
  8. Operator ends the recording once the sequence is complete.
  9. Review quality and repeat if needed.

Key Rules

  • Stay inside the marked bounding area.
  • Keep the target object visible.
  • Use slow, natural, consistent movements.
  • Avoid jerky or fast movement.
  • Wear fitted clothing and remove badge.
  • Avoid loose clothing, shorts, skirts, dresses, high heels, and chunky footwear.
