EGO Data Collection Overview

This overview explains the shared responsibilities between the operator and data collector during EGO recording.

Before Recording

  • Operator checks that the head-mounted camera view is clear.
  • Operator confirms hands, wrists, and task area are visible.
  • Operator confirms no other person or extra hand is visible in frame.
  • Data collector prepares to start with the required hand pose.

During Recording

  • Data collector holds the start pose for 3 seconds.
  • Data collector performs the task slowly and naturally.
  • Data collector keeps movements smooth, deliberate, and consistent.
  • Operator watches that hands, wrists, objects, and task interaction remain in frame.
  • Operator should flag or redo a sequence if important parts leave the frame.

After Recording

  • Data collector returns to the start pose at the end.
  • No 3 second hold is required at the end.
  • Operator ends the recording when the sequence is complete.
  • Operator confirms whether the recording is usable.

Quick Reminder

Good EGO data should be stable, clear, natural, and easy to understand from the head-mounted camera view.
