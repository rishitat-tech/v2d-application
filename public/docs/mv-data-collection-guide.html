MV Data Collection Guide

This guide covers both the data collector and operator responsibilities during MV data collection. It explains where the data collector should stand, how the task should be performed, what should be worn, and what the operator should check during recording.

Before Starting

  • Verify the calibration video is good quality before starting data collection.
  • Use only the required pre-scanned object for the collection.
  • Make sure the object is added to the correct tracker/category if required.
  • Confirm the data collector and object are inside the marked bounding area.
  • Make sure the target object is not blocked from any camera view.

Data Collector Responsibilities

  • Start inside the marked bounding area.
  • Stay inside the bounding area for the entire sequence.
  • Move slowly, naturally, and consistently.
  • Avoid fast, jerky, or sudden movements.
  • Keep the object visible during the task.
  • Follow the action description for the assigned task.
  • Notify the operator when the sequence is complete.

Each sequence is expected to be about 30 seconds. Some variation is acceptable, but following the action description takes priority.

Operator Responsibilities

  • Complete the required recording fields such as Author, Title, and Location.
  • Use the task title that matches the dataset/task being collected.
  • Start recording before the data collector begins moving.
  • Give a 3 second countdown before signaling the data collector to start.
  • Watch that the data collector stays inside the bounding area.
  • Watch that the object remains visible and is not blocked from camera views.
  • End the recording once the data collector says the sequence is complete.
  • Communicate clearly when the session has ended.

What the Data Collector Should Wear

Wear:

  • Fitted clothing.
  • Comfortable footwear that allows safe, controlled movement.
  • Hair tied back or kept above shoulder length if needed.

Avoid:

  • Baggy clothing.
  • Loose tops.
  • Shorts.
  • Skirts.
  • Dresses.
  • High heels.
  • Chunky footwear.
  • Scarves, loose accessories, or anything that can block the view.

Badge Requirement

  • Always remove the badge before starting the sequence.
  • Do not keep the badge hanging or visible during the task.
  • Remove anything that may swing, block the view, or interfere with capture quality.

Good Data Collection

  • Data collector stays inside the bounding area.
  • Only the required pre-scanned object is present.
  • Object stays visible from start to end.
  • Movements are slow, natural, and consistent.
  • Lighting is even and not harsh.

Bad Data Collection

  • Data collector goes out of the bounding area.
  • Loose clothing, hair, or accessories block the view.
  • Movements are jerky, fast, or unnatural.
  • Lighting creates harsh shadows or glare.
  • The object changes shape during capture.
  • Extra objects are present in the scene.

Quick Checklist

  • Calibration video checked.
  • Correct pre-scanned object selected.
  • Data collector inside bounding area.
  • Object visible from camera views.
  • Badge removed.
  • Fitted clothing worn.
  • No loose clothing, shorts, skirts, dresses, high heels, or chunky footwear.
  • Operator starts recording first.
  • Operator gives 3 second countdown.
  • Sequence completed with slow, natural movement.
