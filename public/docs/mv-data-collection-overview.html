Data Collection Overview

This overview explains the basic MV data collection flow and the shared responsibilities between the data collector and operator.

Purpose

MV data collection captures task sequences using the required pre-scanned object. The goal is to collect clean, high-quality data where the data collector, object, and task interaction stay visible and inside the marked bounding area.

MV Setup Example

Before Recording

  • Verify the calibration video is good quality.
  • Confirm the correct pre-scanned object is being used.
  • Make sure only the required object is present.
  • Confirm the data collector is inside the marked bounding area.
  • Make sure the object is visible and not blocked from camera views.

Start Position Example

During Recording

  • The operator starts recording first.
  • The operator gives a 3 second countdown before the data collector starts moving.
  • The data collector performs the task slowly, naturally, and consistently.
  • The data collector should stay inside the marked bounding area.
  • The object should remain visible throughout the sequence.

After Recording

  • The data collector notifies the operator when the sequence is complete.
  • The operator ends the recording session.
  • The sequence should be reviewed for quality when needed.
  • If the collector leaves the bounding area or the object is blocked, the sequence should not be used.

Lighting Requirements

Lighting directly affects MV data quality. Good lighting helps the cameras clearly capture the data collector, object, bounding area, and task interaction. Poor lighting can make the object harder to see, create shadows, increase blur, and reduce recording quality.

  • Make sure the area is well lit before recording.
  • Avoid dark rooms, strong shadows, glare, or uneven lighting.
  • Confirm the object and person are clearly visible from the camera views.
  • If lighting is poor, fix it before starting data collection.

Key Reminders

  • Stay inside the marked bounding area.
  • Keep the target object visible.
  • Use slow, natural, consistent movement.
  • Avoid fast or jerky motion.
  • Wear fitted clothing and remove badge.
