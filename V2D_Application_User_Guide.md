# V2D Application User Guide

## Overview

V2D is a resource hub for MV, EGO, Object Scanning, Dashboard User Guide, and training quizzes.

Use this application to find guides, videos, examples, checklists, FAQs, and quizzes in one place.

## Main Sections

The application includes these main sections:

- Home
- V2D Resource Hub
- MV
- EGO
- Object Scanning
- Dashboard User Guide
- Quiz

## Home Page

The Home page welcomes users to V2D and provides a button to open the Resource Hub.

Use the Home page when you want to start from the main entry point.

## V2D Resource Hub

The Resource Hub is the main place to access all training and reference materials.

From the Resource Hub, users can open:

- MV
- EGO
- Object Scanning
- Dashboard User Guide
- Quiz

Use the search box to quickly find a section.

## MV Section

The MV section contains resources related to MV workflows.

MV resources may include:

- Overview documents
- Videos
- Documentation
- Good examples
- Bad examples
- Checklists
- FAQs

Use the category tabs at the top of the MV page to switch between resource types.

### MV Overview

The Overview tab gives high-level MV workflow information.

Use it to understand the general MV process before reviewing detailed documents or examples.

### MV Videos

The Videos tab contains MV training and example videos.

Videos open in an in-app video player.

### MV Docs

The Docs tab contains MV documentation and guides.

Use these documents for detailed instructions about calibration, data collection, operator responsibilities, and data collector expectations.

### MV Good Examples

The Good Examples tab shows examples of correct MV data collection.

Use these examples to understand what a good sequence should look like.

### MV Bad Examples

The Bad Examples tab shows examples of issues to avoid.

Use these examples to learn what can make a sequence unusable or low quality.

### MV Checklist

The Checklist tab provides quick reminders before and during MV data collection.

### MV FAQ

The FAQ tab answers common MV questions.

The Dashboard User Guide section provides access to the dashboard guide.

## Quiz Section

The Quiz section contains training validation quizzes.

Available quiz sections may include:

- MV Quiz
- EGO Quiz
- Object Scanning Quiz

## Taking a Quiz

To take a quiz:

1. Open the Quiz section.
2. Select the quiz you want to take.
3. Enter your name.
4. Enter your email.
5. Answer every question.
6. Click Submit Quiz.
7. Review your score.

After submitting, the app shows:

- Score
- Percentage
- Selected answers
- Correct answers for missed questions

## Saved Quiz Results

Quiz results are saved in the browser.

The Saved Results table shows:

- Quiz name
- Name
- Email
- Score
- Percentage
- Submitted time

Results are only saved on the current browser/device.

## Downloading Quiz Results

Use the Download CSV button in the Quiz section to export saved quiz results.

The CSV file can be used for review or tracking.

## Searching Resources

Each resource section includes a search box.

Use search to find resources by:

- title
- description
- tags

## Opening Resources

Depending on the resource type:

- Documents open in a new tab.
- Videos open in the in-app video player.
- Images display directly in the card.
- 3D models display directly in the card.
- Checklist and FAQ cards display information directly without needing to open anything.

## Resource Status

Some resources may show as unavailable if files or links have not been added yet.

If a video or media file is missing, the card may show:

Media file not added yet.

## Recommended Usage Flow

For new users:

1. Start at the Home page.
2. Open the V2D Resource Hub.
3. Select the relevant section: MV, EGO, or Object Scanning.
4. Review the Overview tab.
5. Open the Docs tab for detailed instructions.
6. Review Good Examples and Bad Examples.
7. Use the Checklist before starting work.
8. Take the related quiz.
9. Download quiz results if needed.

## Notes

- Use guides for detailed instructions.
- Use examples to understand quality expectations.
- Use checklists as quick reminders.
- Use quizzes to confirm understanding.
- Quiz results are stored locally in the browser.
